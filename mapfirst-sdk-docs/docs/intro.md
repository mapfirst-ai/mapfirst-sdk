# Welcome to MapFirst SDK

MapFirst SDK is a powerful mapping solution that provides intelligent property search with AI-powered filters across multiple map platforms.

## üöÄ Key Features

- **Multi-Platform Support**: Works seamlessly with MapLibre GL JS, Google Maps, and Mapbox GL JS
- **Smart Search**: AI-powered search with natural language queries
- **Interactive Filters**: Dynamic filter chips with real-time updates
- **React & HTML**: Full support for React applications and vanilla HTML/JavaScript
- **Responsive Design**: Mobile-first, adapts to any screen size
- **Accessible**: Full keyboard navigation and ARIA support

## üì¶ What's Included

- **@mapfirst/core**: Core SDK with map adapters for MapLibre, Google Maps, and Mapbox
- **@mapfirst/react**: React hooks and components including SmartFilter
- **HTML/Vanilla JS**: Use directly in any HTML page without a framework
